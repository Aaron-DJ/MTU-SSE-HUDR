# MTU-SSE-HUDR
Arduino code for the Heads Up Display used in MTU SSE Car
Required libraried: SdFat, NeoGPS, NeoHWSerial
Setup the NeoGps cfg to run off of NeoHWSerial and set it to use interrupt
